First web app

In this page there will be a brief explanation of what each tab does on the menu to the left.

Main Page - Returns you to this very page.

Register - Register a new User by filling out the necessary parameters.

Login - Once registered you can log in with your ID and Password.

Role - Change the Role of a User (behaviour depends on your own Role).

State - Change the State of a User (behaviour depends on your Role).

Remove User - Remove a User from the Database (behaviour depends on your Role).

List Users - List all the Users you have permission to see (behaviour depends on your Role).

User Attributes - Change the various Attributes of a User (behaviour depends on your Role).

Change Password - Change the Password of a User (behaviour depends on your Role).

Your Token - Displays your Authentication Token for this current session

Logout - Log out of your account.
